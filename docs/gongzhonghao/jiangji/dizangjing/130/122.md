# 地藏菩萨本愿经讲习 第一百二十二讲

地藏菩萨本愿经讲习

感恩地藏菩萨加被，我们接着讲。

> 主产鬼王、主命鬼王、主疾鬼王、主险鬼王、

这里说到这个`主产鬼王`是干什么的呢？

这个`主产鬼王`主要是管理`鬼道中在人或者畜生生产时候出现的小鬼`的，鬼道中有一些小鬼，这些小鬼在人或者畜生道的牛马猪等生产的时候，就聚集过来了，他们是来吸食脓血，生产肯定是有血乃至有胎盘的，他们对于产血也是`嗅`一样，然后就依如来藏变现在他们手上了，然后他们就吃了，他们`嗅`过的就不能再`嗅`，因为他们吃过了在他们的世界就不存在了，在他们看来是没有了（福德多一点的鬼会在他的世界会发现更多）。当然你还看得到的所以还有胎盘等存在的，因为我们跟他们业报不同世界不同的（虽然世界不同，但入处饿鬼的世界和人道的世界重叠，有些特殊的鬼可以利用人道的比如排泄物等变缘他所受用的食物等）。鬼道众生都有统帅都有主人，这些鬼就归这里的主产鬼王管理的，主产鬼王就是他们的主人，世间所谓孤魂野鬼，在鬼道没有谁愿意当孤魂野鬼，那样就意味着经常被欺负，连浓痰都吃不到，鬼道有鬼道的次序的。

**主命鬼王。**

`主命鬼王`是干什么的呢？我们人活着的时候是遇不到他的手下的，人将要死掉的时候，就会遇到他的手下了，这些鬼来干什么呢？来招引的，搞`接待`工作的。比如有些人要死掉了，会做梦梦到有人来敲门，门开了，外面停着华丽的车马，上面坐着自己已经去世的眷属，已经死去的眷属说，好想你啊，走啊，跟我们去玩啊！我们好好聚一聚啊，这一去就在人间死掉了，往生鬼道了。或者是自己漂浮在天花板上，看到所谓的天使来迎接自己了，这种事情一出现，往往就意味着人道的寿命要结束了，在前面也讲了很多了，的确是有这个事情的，有些人不晓得什么原因，遇到过后来可能福报还没有结束，或者说别的什么原因，又回来了，又或者定中梦中看到过，于是就认为有所谓的`黑白无常`，这些鬼都归主命鬼王管理，主命鬼王是他们的统帅，他们的主人。

**主疾鬼王。**

凡是经常看佛经的大家就晓得，这个人生病，主要是三种原因。第一是因为`业报`导致的生病，比如我那位女性朋友，我刚到上海来的时候对我很照顾的，因为先世是杀鸡鸭的，杀鸡鸭杀掉之后血还没有流完，赶紧掏空肠子拔毛等，这个时候是最好拔毛和掏肠子的，你等久了血液凝固了，肉体僵硬了就不好弄了。她那一世是杀鸡鸭的，果报都早已经实现过了，这一世还有余报，得了`绿色肠癌`就是`克罗恩病`。这个病不产生癌细胞复制，但是一直炎症不断，不明原因的炎症导致肠粘连没法消化，她肠子都被割了五六次了，肛门都割掉了....哎，我除了捐钱，就是把功德回向给她，好像最近没有复发了，希望一直能够如此（疫情放开之后，得了新冠又去做手术了，哎），所以业报导致的真的很惨痛的。

第二种是`四大不调`，我们群一位师姐，得了抑郁症，其实她得这个病即有先世的业报导致的缘故，可能曾经有一世是皇室贵族，这些人没有事干，吃得好玩得好，喜欢打猎，导致这辈子还有余报得了抑郁症，当初那些动物多惊恐，多烦躁啊，报应啊。然而她自己也有原因，从来不按时吃饭，不按时睡觉，举要言之，生活不规律引起四大不调，其实就先世来说，影响已经不大了，她偏偏生活作息不规律就导致抑郁症反复发作，如果能够老老实实的`念诵地藏经拜忏`，加上`生活规律`，完全可以不久康复的（后来我终于搞清楚她为啥会得抑郁症反复发作治不好了，哪怕帮她回向都不行，因为她表面看是信佛的结果是压根不信因果的，自己有房有车还跑到正法群来众筹借钱买保险，这个因果她都敢担待！还早些年去很多寺庙当义工，她不信因果估计跟在寺庙当义工也有关系，因为她看那个寺庙的出家人都不信都大妄语。所以说我建议学佛现在也别东奔西跑，我都遇到好几个这样的了，本来好好的，结果去寺庙转转，发现出家人都这样的，而放弃学佛了，其实她这个既有业报的缘故也有自身生活不规律的缘故）。

第三种是`鬼神`导致的疾病，比如说某人一直晚上做噩梦，噩梦做多了睡眠不好，心脏，肝脏，脾脏等没有办法休息。这人身啊，就是一架机器，天天睡眠都不好，这架机器耗损就大，久而久之就生病了。推敲她为什么老是做噩梦呢？她各大医院都去看了，没有什么原因，`查不出来`，久而久之`神经衰弱`乃至`周身是病`。其实这个时候就要考虑是鬼道的一种鬼，这种鬼喜欢作弄人，叫做`魇鬼`，魇鬼就是喜欢在`睡梦`中捉弄人的，当然能够导致人生病做噩梦的可不止这一种，还有很多种比如`补单那鬼`，这种长得像个`泡菜坛子`的鬼，也喜欢捉弄人，让人做噩梦让人生病，身体虚弱精神恍惚的，不管多少种，这些都归这里的主疾鬼王管理，`主疾鬼王是他们的统帅`，`他们的主人`（我接触过很多鬼神导致疾病的，比如我前任就是，还有这几天接触一个是小时候经常梦到恐怖的鬼怪，后来经常生病，并且几十年都不敢一个人单独在房间睡觉，就在身上纹了`迦楼罗`的像，她认为这样可以免除噩梦导致的疾病）。

这里的`主疾鬼王`就是负责`管理`这些可以`导致人道乃至畜生道生病的鬼神`的。

**主险鬼王。**

有些鬼喜欢在深渊，在高山峻岭，在危险的地方使坏的。如果谁做了恶事，或者谁福报快到头了，当然他是不晓得他福报快到头了的，偏偏他无明的缘故，不信因果的缘故，平时也不积福。又或者说就是喜欢刺激，有钱就是任性，我年轻勇于挑战，去玩什么漂流啊，去蹦极啊，去旅游啊，到比较危险的地方去显摆，就会遇到这种在危险地方的鬼。这种鬼看到他在漂流，本来他正在把握方向的紧要关头，这种鬼跳出来，吓他一下，完蛋了船翻掉了，头撞到旁边的岩石上了死掉了。当然也没有人告诉大家有这种鬼，因为看到的十有八九都死掉了，不该死的也看不到（其实我觉得旅游，有什么好旅游的，在家诵读佛经有哪儿不好吗？你看来看去不都是看的自己的`内相分`嘛，都是你自己如来藏为本质而变现的啊，`虽有非真`嘛）。他们还能够迷惑人，比如去问路，他使得说路线的人说错，把人往险恶的地方引，还能够在晚上或者是平时使人迷路，导致行路的人走来走去怎么还在这个地方啊！俗话叫做`鬼打墙`。这种鬼的统帅就是`主险鬼王`，`主险鬼王是他们的主人`。

**三目鬼王。四目鬼王。五目鬼王。**

听名字大家都不晓得是啥意思，模模糊糊就认为一种鬼王长了三只，四只，五只眼睛，我这么说吧，这个名字大家应该听说就是`马王爷`！有句俗话叫做，你把我惹着了，我今天让你晓得`马王爷有几只眼`！这么说就是要让你好看！其实马王爷在鬼道`不止一位`，有好多个的，有三只眼睛的，有四只眼睛的，有面上有五只眼睛的（以后大家就可以回答，有三只的，也有四只，可能也有五只眼的），`道教`称为`水草马明王`，这个马王爷过去是战将，过去打仗都是骑着马打仗，他勇猛异常，保家护国，后来战死了，就往生到鬼道当了马王爷了，他的形象还是骑着战马的，赤面虬髯，执刀持戟，面目狰狞，鬼道大多数的鬼乃至罗刹都很怕他的（据说人间有大灾难的时候，他也`负责运送亡神`的，阴兵借道就属于运送亡魂，他就负责把亡魂其实是中阴身运送到鬼道去，防止其他鬼道的骚扰，比如以前`唐山大地震`的时候就有人遇到`阴兵借道`的事情，他一般是在大灾难之后很快就会来护送亡神的，他在前面骑着高头大马，后面跟着大队的阴兵，再后面就是牛车，马车拉着很多的面容惨白，哭哭啼啼的亡神）。

**祁利失王。大祁利失王。**

`祁利失`是梵语，翻译过来就是`大力士`。鬼道有一种鬼王`有很大的力气`，力气大到可以`搬山`，他管理的鬼各个力气都很大。过去有种咒语叫做`搬山术`（据说`鲁班全书`里面还有这个咒语的记载，我没记错应该是在下册，但下册最好不要看，一来你跟他们有缘证明你八字弱，二来有缘的基础上修这些容易惹灾祸）。我记得小时候我们老家的一个山顶上有很多大石头，周边很远都没有这种大石头，一颗石头大概几十吨，据老年人说就是有人用搬山术，请了祁利失王的手下（当然中国民间不是叫这个名字，一般都是说`大力鬼`之类的）。有咒术请`大祁利失王管辖的大力鬼`来移动这些石头，但请这些鬼神必须要念这个咒语的，念了之后，这些大力鬼就来搬运，几十吨的石头都可以搬动的。这个人用了这个咒语要把这些大石头搬到某地的，结果祁利失王的手下干活儿太卖力搬得太快，会这个咒语的人在后面追，追到前面遇到一个神祗，就问，老头儿，你看到我前面赶的`猪`没有？`老头儿`说，我没有看到，我只是看到了一堆石头！咒语就不灵了，被这个`神祇`给破了，再祈请也没有作用了，这些石头就在那个山上不动了。

**祁利叉王。大祁利叉王。**

`祁利叉`也是梵语，翻译过来就是`火`，这其实就是`火神`。鬼神道的火神，这个火神是干什么的呢？顾名思义是`专门掌控火的`，过去烧陶瓷啊，烧砖头啊，炒菜啊，都需要火不能够熄灭，这样保持高温不断的燃烧，开了新的砖窑啊，都要祭拜他的，祭拜他之后，他下面的小鬼神等可以使得这个砖窑一直比较旺，砖窑不断火生意好，烧出来的砖头好，烧出来的陶器好，现在工业化的，有些工厂炼钢，有些砖窑，有些陶瓷厂，做出来就是不好，其实拜一拜这位祁利叉王。大祁利叉王。也挺好的啊，凡是跟火有关的产业，跟他多少都有点关系。另外现在火灾也很多的，像他下面的小鬼都喜欢玩火，福德少的人遇到这些很容易遭火灾，`念念祁利叉王`。`大祁利叉王`。他们晓得你认识`他们管事儿的`，就不敢放肆了。

后面这个`阿那吒王`。`大阿那吒王`。比较重要，属于重量级鬼神，所以留待明天细细的讲，今天暂时讲到这里。
