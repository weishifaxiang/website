# 地藏菩萨本愿经讲习 第一百四十七讲

> 主讲人: 内院看门人 <br />
> 校订: 李师兄，胡师兄，利行师兄 <br />
> 审核: 胡师兄，利行师兄，孙师姐，黄兰师姐 <br />

好，我们现在接着昨天的继续讲!

> 是阎浮提行善之人。临命终时。亦有百千恶道鬼神。或变作父母。乃至诸眷属。引接亡人。令落恶道。何况本造恶者。

接着`主命鬼王`说，这个阎浮提就是我们这里的众生，即便是行善之人，这里的行善肯定`不包括佛教证果`的人，`佛教真正证果的人鬼神道的是惹不起的`，我们平时看不出来，俗话说证果两个字也没有写在脸上，如果他还愿意继续往生人间的话，他舍报之后哪怕就是证初果须陀洹的人，一般的恶鬼神都不敢靠近的，其一毕竟的有很多的`善鬼神拥护`他的，其二毕竟鬼神道的看得到证果的人所产生的`光是不同的不敢靠近`，所以没有下面这些情况。这里说的行善之人，是说的世间善行的，或者学佛还没有证果的，就说这样的行善的人，他们在临命终的时候，也有很多的鬼神变成他们眷属的模样，比如他的亡妻他的亡夫，或者已经逝去的父母等等。

变成他的眷属的模样来干什么呢？来接他去鬼道中，为什么要接他去鬼道呢？因为不论是人还是天人，还是`鬼道的鬼神`都希望自己的`势力更加庞大`的，都希望自己家大业大的，把这些的人迎接到鬼道来的话，自己的势力就会更大，好比现在说的抢人才，鬼道不同势力的鬼之间经常斗争的，自己这一方人员众多，当然对他说是好事的，所以他们晓得谁要临命终了，就运用五通观察这个人的亲友已经逝世的是谁？或者思念谁？如果是思念自己逝去的爹妈，就变成爹妈的形象，如果是思念自己逝去的妻儿，自己逝去的老公，就变成妻儿老公的形象，当然这样只能够是`形似而无法神似`。毕竟跟自己那么多年性格语言等没法变，不过人之将死，是不容易察觉到这种不同的，跟他们走了就完蛋了，跑到鬼道去当有财鬼了，想要脱离那可就难哦！这变化的都是假的，因为他真正的父母什么的难道一定就下鬼道了？这就不一定，而且鬼道的条件也不好，即便下了也不会想着拉他去。

你会说，还有这种事情？变`天使`的都有的！看到天使来迎接自己，西方不是有`濒死体验`嘛，他们这些鬼神都有`五通`，都是随将亡之人的爱乐来变化的，比如信基督的，他们变成天使不就更容易拉这个人下鬼道嘛！有人后来活过来说，看到`自己的爹妈`，看到`天使`来了来迎接自己了于是有所谓的体验。比如前几年我看了东北的一个故事，是上世纪八十年代的，有个孩子生了重病当时是军垦，所以离医院很远，她老爸开车就送医院去，在送去的过程中，这个孩子就看到一辆车，上面有七八个孩子`好多玩具`，跟他老爸的货车是并行的，叫她跳过去玩儿，她就跳过去，结果发现到了一个黑暗的地方，周围都是面容狰狞的小怪物一样的，她听到啪的一声，又活过来了，原来是医院在电击她...

那么你会说，这段经文讲生前是行善的咋会出现这样的事情呢？每一个众生都有`如来藏`，如来藏第八识也就是阿陀那识是常住的，所以导致生生世世有很多的善恶业种的，有些已经实现了，有些还没有实现的，这辈子活着是正在实现某些种子，比如说某人上一世做了一件恶事，按照道理这辈子是要`下堕畜生道`的，但是他上上一世有某件善事，这辈子有合适的缘，也就是条件先实现了上上一世的善事的善报所以成为人还成为富人。这辈子并没有下堕三恶道。这辈子固然也行善，可是他没有学佛，当然也`不会忏悔上一世的恶业`，因为他不学佛根本不信因果轮回。上辈子那件恶事导致的恶报，就在下辈子出现，导致他堕落鬼道，所以有这些恶鬼神变化他`眷属勾牵他去饿鬼道`的事情出现的，总而言之如果不学佛，不学正法（不学正法是绝对不会对于因果轮回深信的，因为学正法才会坚信`如来藏心体不灭`，`受熏持种`）就意味着`不会忏悔`，`断不了自心流注`，那么就会遇到这样的事情。他这辈子行善，要在下辈子堕落到鬼道结束之后，再到人间来实现，或者说在鬼道成为有福鬼来实现的。
