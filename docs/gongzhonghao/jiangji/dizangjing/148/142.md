# 地藏菩萨本愿经讲习 上 第一百四十二讲

> 校订: 李师兄，胡师兄，利行师兄 <br />

地藏菩萨本愿经讲习

感恩地藏菩萨加被！

> 便令卫护。不令恶事。横事。恶病。横病。乃至不如意事。近于此舍等处。何况入门。

昨天我们讲了这个`恶毒鬼王`，也就是示现在鬼道的`菩萨`说，如果有人类中的人，能学习`阿陀那识正法`或者说`如来藏正法`，也就是佛法。佛者，觉也！觉悟是说的`相应于实相`才叫做觉悟，实相在此世界是如此，在彼世界还是如此，在欲界是如此，色界，无色界都是如此的才叫做实相。实相必然是有所依的，必然是有个东西才叫做实，不能说无常就是实，无常是对于现象界的总结，`无常就非实`（比如你现在二十，以后四十，如果二十是实，那么四十呢？毕竟三十和四十思想身体差别这么大，实就不变，变化就不是实），说实相就是说的所依的真心如来藏是实，由`亲证这个非无常非非无常的如来藏`才有实可言。我前段时间看知乎上有个学哲学的说`佛法`是`诡辩论`，因为他接触到的所谓`佛法`讲无常就是常，那么他说既然无常就是常，就没有常啊！因为常就是变动，变动还有个什么常？常就不变，变就不常嘛！他说得颇有道理。我们大家可不要成为`诡辩论`啊，需要知道佛教是依实相心如来藏而建立的，`如来藏或者叫做阿陀那识其中种子生现行`，`现行熏种子是刹那生灭`，`而心体是常住的是真实的是永恒的`，`由亲证阿陀那识也就是藏识故`，所以叫做`亲证实相`。

能够宣传十二分教，依本际如来藏不灭，去跟人家讲佛教的`故事`，比如`莲华色比丘尼`的故事，`目犍连尊者`的故事，乃至佛教的`譬喻`，比如`盲人摸象`的譬喻（这个譬喻是说的世间外道如盲人一般的不见真实，执蕴处界为真导致所说都是颠倒错乱的），盲人骑瞎马夜半临深渊的譬喻（其实这个譬如就是说的末法时代的邪师很多，如我前面讲的那些邪见，他们也四处去宣讲他们的，真是一盲引众盲），乃至把佛教的理论，所谓`业杂染`，`生杂染`，`烦恼杂染`讲讲，讲`因果报应之所以成立`的理由等等等，这样的话恶毒鬼王等才会敬礼这个人，你就是未来佛啊！不应该敬礼吗？并且让恶事，横事，恶病，横病不会出现在这个人身上，不会找到这个人。当然你会说，不对哦，也有宣传如来藏正法而还是有恶病横病的啊，是有的，但那是因为`先世罪报`，导致今生出现这些病症，先世罪报是为什么会来到今生？还不就是因为阿陀那识受熏持种嘛，正因为阿陀那识受熏持种所以导致今天还有这些病症出现，不过其一，就算`没有开悟也晓得有本心不灭`，心理上当然不会是凡夫那样惶恐不安是安乐的，就算业报所致也慷慨赴死，毕竟有本心不灭，所谓生死于我如浮云。其二，因为宣传正法的关系，尽管有先世罪报，也可以`少受很多罪`，寿命延长很久，比如我们群有师兄十几年前就查出来癌症，到现在都好端端的，平常看起来也并无什么痛苦的。

> 乃至不如意事。近于此舍等处。何况入门。

**不如意事**有很多的，我们生在这个世界，衣食住行不求奢华求个普通，也是求啊，这就是有所求，所求不能满足故名不如意。很多人的非分的要求，比如生在一个普通家庭，要当部委干部，要当省委干部，这些不切实际的要求也就不谈。这里是说不过分的要求，其实就普通人来说，很多的正常的要求往往也难以如意，比如都希望身体健康，按理也不过分儿，平时也注意保健，但还是老往医院跑。比如求个三餐饱暖（加班加点才求个温饱显然不能算如意），乃至求一个七八十平方的房子，现在房价这么高，往往也难以如意。其实这个很正常的，在地狱最普遍的苦是受折磨，饿鬼道最普遍的苦是饥寒苦，人道最普遍的苦是求不得苦，所以皇帝总统都不可能事事如意的，你事事如意你不是正常人，你要么是蠢人要么是天人。

如果宣传佛教，所谓**或有男子女人。修毛发善事。乃至悬一旛。一盖。少香。少华。供养佛像。及菩萨像。或转读尊经。烧香供养。一句一偈**。那么这些事情经过一段时间都可以如意，如`恶毒鬼王`这样的菩萨，他有他心通的晓得你的要求，就会运用神通，乃至命令手下的小鬼王，鬼神等协助你满足你的愿望，只要不过分的愿望，大多稍微努力，都可以办到的。

**近于此舍等处。何况入门**。也就是说不单单说他不会遇到恶事，横事，恶病，横病。乃至他周遭，他的家人，甚至他的邻居都不会遇到恶事，横事，恶病，横病（比如你要是有邻居插头冒火烧了房子，你不也倒霉嘛），这些`倒霉事情`离`持地藏经之人`远远的，之所以离他远远的，是恶毒鬼王等有大力神通，其中有天眼通能够看到未来发生的事情，假如未来要出现什么状况，飓风起于青萍之末，他们观察这个倒霉事最开始是因为什么而起的，譬如熊孩子向下面扔花盆，你恰恰走那里过，这些鬼王就干扰一下，熊孩子的玩具突然响起来吸引了他的注意力，他就没扔成，这个被十楼的花盆砸头的倒霉事就不会出现。乃至周遭都不会出现这个事情，已经`防范于未然`了不能近前了，何况他真的遇到呢？可能遇到呢？至于一般的恶鬼神等，包括恶毒鬼王等大力鬼王当然更不会让他们近前了。

恶毒鬼王为什么要这么做呢？对他有什么好处呢？其一因为他本身就是`菩萨`，菩萨的职责就是`护持佛法`，让正法久住。人间没有正法了，三恶道就越来越多乃至充满，人道和天道的人数就稀少，他本身是在鬼道度众生的，把鬼道众生度到哪儿去？首先是度到人间和天道来，人间的持正法的人他保护了，正法宣传得到巩固，使得更多人`不下堕鬼道`，他不就达成目的了吗？你在人间宣传如来藏正法，你帮他大忙了啊。其二，成佛需要很多的`福德`的，恶毒鬼王也想成佛啊，我们也想成佛啊，他帮助你，你是学习正法宣传正法的，等于是他修布施，我前面也说了布施有三种，财法布施和无畏施，他帮助你远离厄难，难道不是无畏施？他是`积累福德`的，他福德积累越多，他成佛越迅速的，他为什么不干呢？并不是他跟谁谁有多熟，关系有多好，佛教的修行上只有公义而无私情，有公义才有真正的私情。

> 佛赞鬼王。善哉善哉。汝等及与阎罗。能如是拥护善男女等。吾亦告梵王帝释。令卫护汝。

恶毒鬼王讲完之后，佛陀就赞叹恶毒鬼王说，说得好啊，说得好啊！善哉，善哉，你跟阎罗天子等鬼神能够如此的拥护修习正法的善男子善女人，他们在鬼道是有大力的，好比人间的领导，领导起了表率作用，下面的人乃至其他的有力量的小鬼神，其一当然也开始`仰慕学佛`了，因为领导学佛嘛，领导要是不学佛，他为啥护持学习正法人呢？学佛肯定好事儿啊，所以他们也会去学，比如领导去吃包子，大家都去吃包子了，上行下效。其二这些小鬼神也会去`护持学正法`的人的，鬼王这么做多少是因为`培植福德`，鬼王这么大的福德都还需要培植，我为什么不培植福德呢？其三，在鬼道这么大神通能力的鬼王都护持这位颂持地藏经的，其他的恶鬼神当然不敢来干扰了，纵使先世和这个学佛人有什么仇怨，暂时也不敢惹了。这样就一举三得了，所以佛陀赞叹，他们这样做，人间的正法宣传者得到护持，鬼道的小鬼神等也能够仰慕佛法，所以连接两个赞叹，善哉，善哉！

**吾亦告梵王帝释。令卫护汝**。你们这么做，我也告诉`梵王`，梵王就是`大梵天`，大梵天是小世界地位最高的，我在前面也讲过的，从光音天堕落到小世界来，最先见到的是大梵天初禅天王，这个小世界要毁灭，最后离开的也是初禅天王，所以大家有造物主情结，认为是他创造的。这些鬼神能够做这样的福德，佛陀说告知梵天能够让他们修习禅定，将来往生梵天，由大梵天王的护持当然更容易成就禅定。乃至告知`帝释`，某某鬼王护持正法，帝释就是`忉利天的能天帝`，属于地居天的最高领导者，如阎罗天子诸大鬼王都接受他的管辖，这些鬼神也想早脱鬼伦啊，环境也不好，吃得也不好，虽然有福德，但一天到晚小鬼在面前打打杀杀的，或者饿得嗷嗷叫的声音不绝于耳（虽然鬼王很安乐，但环境不好），日月无光愁云惨雾的，他们当然愿意往生天界了（除大愿在鬼道度众生的菩萨），而且因为帝释天听佛讲这位鬼王护持正法，`帝释天`当然对他青睐有加，他的势力就越来越大。所以这里说**吾亦告**。
