# 地藏菩萨本愿经讲习 第一百四十五讲

> 校订: 李师兄，胡师兄，利行师兄 <br />
> 审核: 胡师兄，利行师兄，孙师姐，黄兰师姐 <br />

地藏菩萨本愿经讲习

感恩地藏菩萨加被！

> 或已生下。慎勿杀害。取诸鲜味。供给产母。及广聚眷属。饮酒食肉。歌乐弦管。能令子母，不得安乐。

昨天我们讲了孩子在初生的时候，在未生的时候但凡能够做`种种善事`，特别是有益于佛法传播，正法传播的善事，那么子母都能够安乐。这里说孩子生下来之后，我们一般的人就会说，生下来之后要给孩子补充营养，要给母亲补充营养，所以就`杀害生命`，比如说种种鲜味，所谓鲜味比如说的山珍海味，鹿茸乌龟甲鱼等等的是杀生而来，杀害生命而来，乃至说把亲戚朋友都叫过来，办什么满月酒之类的。

我女儿满月我前任强烈要求办满月酒，她主要是希望亲朋好友都拿钱然后放自己的口袋里面去。虽然吃的都是三净肉，这个肉类并没有看到动物被杀，并没听到动物被杀的惨叫，也不是说我要办满月酒，所以酒店专门为我杀的。结果也是不得安宁的，我前任请来了一个她网上认识的闺蜜，她这个朋友到我家吃住好长一段时间（长得好胖起码三百来斤是有的），我母亲有点微词，闹得很不开心，我觉得的确是不得安乐的。**饮酒食肉。歌乐弦管**不单单说取诸鲜味，还为了庆祝的缘故，请很多人来办酒席，办酒席在过去当然要弹奏乐器，搞娱乐节目的，以前请人来唱戏，现在是请人唱卡拉OK的唱歌的，这样都会导致母子不得平安，我发现我家那段时间的确是`不安乐`的。

`杀害生命而长养自己的生命`，这`很不好`的事情，从因果报应的角度来说，正因为`一切众生皆有如来藏第八识`的关系，所以不论你是怎么样的目的这个`杀业成就了`，那怕你我并没有亲自去杀，请的厨师去杀，厨师是因为你要办酒席或者什么原因而杀，那么你们就共同成就杀生业了。这个`众生之间如来藏是互相为缘`的，我前面也多次强调了，`乔布斯`发明`智能手机`，是他个人如来藏中种子的现行，不然为啥别的人发明不了？他如来藏中种子现行导致智能手机被发明出来，也就给`其他人的如来藏中种子现行提供了条件`了，于是大家都有智能手机用，小孩出生父母办酒席叫厨师去杀生，厨师去杀了，这个事情完成了，厨师的如来藏受到熏习，父母的如来藏也受到熏习，小孩的如来藏也受到熏习，因为他们是共同完成，共同配合来完成杀业的。`未来就有恶报`的。

很多人也许看到这里会说，某人作恶神不知鬼不觉如何，首先`不可能神不知鬼不觉`的，干啥他们都知道的，我们在他们眼里就是动物园被关在铁栅栏里面的动物，他们看得一清二楚的。其次你会说，某种事情比如特殊时期共同造业，将来受报不差分毫，这是怎么计算的？某师姐最近天天跟我纠缠这个问题，就好像我们吃喝拉撒需要计算吗？我们吃饭一口接着一口，有谁认真的计算了一口吃多少粒米？一口接着一口要多少秒？没有啊！那你吃到饭了吗？吃了啊！如来藏中种子现行也是如此，不需要计算，有那种业在将来实现，其他人的业报现行了，导致你的业种也会相应的现行的，因为每一个众生如来藏中有无量无数的种子善恶都有，比如父母造恶的关系，固然是父母受，但是父母受恶报比如破产了，这肯定是恶报，导致孩子受了牵连婚姻失败，流落他乡，打工自活。不能说父母的恶业报应在孩子身上，但是`众生的业报是互相为缘`的，父母的恶报实现，导致孩子的恶报也就实现了，为孩子恶业种子现行创造了条件，从这个道理讲，**积善之家必有余庆，积不善之家必有余殃**是有道理的。

> 何以故。是产难时。有无数恶鬼。及魍魉精魅。欲食腥血。

主命鬼王接着说，不论是在将要生孩子还是怀胎的时候，或者已经生下来之后（坐月子时期）不要杀害生命取诸鲜味，乃至说广造杀业。何以故，就是说为什么？因为在生产的时候，这里用了个`产难`，为什么要说是产难呢？产难就是现在说的难产，主要是因为母亲骨盆腔狭窄、子宫或阴道结构异常、子宫收缩无力或异常等造成胎儿分娩困难，所以一直把生产的过程称为产难，是母亲和孩子的灾难关头，稍有不慎就可以导致悲剧。

本来这个生产，生孩子的过程就是个灾难，能不能顺产乃至母子平安就要看`福德多不多`，家庭有没有福德，在这样的情况下，生产的时候还有`无数的恶鬼`，一些`魍魉`和`精魅`这样的恶鬼，这些属于小鬼怪不属于鬼神，这些饿鬼道的小鬼怪就在生产的妇女旁边等着，准备吃`腥血`，吃这种`恶露`，这些`污秽`的东西，对于他们这些恶鬼来说是丰盛的美餐，美味佳肴，你会说，咦，不对吧，我看生产的时候，腥血等并没有减少啊。其实这些小鬼的吃，在我们说来用`嗅`更准确一点，他们接触到这个`食物`，如果他们抢到了吃上去，他们的如来藏就会把这个腥血变化在他们的手上，他们就可以食之，我们看见恶露是没有减少，他们`嗅`过的，在他们看来就没有了。而我们还看得到，因为我们跟他们业报不同，所见不同的，也就是说人类的腥血等做了饿鬼道这些众生食腥血的增上缘，他们的如来藏依人类的腥血变化了一个相似的在他们手里，导致他们可以吃，而人类这边的并没有消失，是因为人类这边的如来藏还在不断变化的关系。而我们讲人道的这个蛋糕你吃了张三没得吃，是因为你和张三都是人类的，有业报实现的直接关系，而你跟饿鬼道的这些鬼是业报实现的间接关系。

他们这些鬼怪因为喜好腥血的关系，对于腥血的气味非常敏感很远都闻得到，就早早的在医院或者说将来生孩子的家里待着了，他们是看得到人类的。就好比说很多跳大神的久久熏习鬼神法，不必有神通，也能够感应到鬼神甚至偶尔看得到，他们这些小鬼深切希望`美餐`的关系，所以看得到生产的妇女乃至孩子的，来得有很多这样的小鬼，而鬼道的福德差，`依人道的腥血粪便等为增上缘`，他们自己的如来藏变现的却很少，所以往往很多小鬼吃不到，吃不到就很愤怒，好不容易有机会美餐一顿，结果被别的小鬼抢了，很愤怒就会迁怒在生产的妇女乃至出生的孩子身上，就对于`母子不利`（比如有些小鬼就钻进孕妇的肚子拉扯孕妇的大肠，或者咬婴儿的手脚，导致难产）。如果你福德多，福德在鬼道众生来讲某人多不多他们是看得到的，人间说的阳气旺的就叫做福德多，一般遇不到鬼，也是这个道理，要是这个家庭，这对母子福德少，就麻烦了就要成灾难了。

> 是我早令舍宅土地灵祇。荷护子母。使令安乐。而得利益。

主命鬼王接着说，他在鬼道示现，他本身又是`大菩萨`的关系，对这个事情当然是非常了解非常清楚的，既然是主命鬼王当然护持众生的生产也是他的本职工作，但凡是跟他有缘的（没有缘就很难了，比如你是外道的这就没有缘，就要靠你自己福德多少了，既然没有学佛你们第八识之间在当下没有办法建立这种互相为缘的关系，比如你是`非洲的土著部落`，你暂时是使用不了乔布斯发明的智能手机的，或者是学所谓`喇嘛教密宗`的人，实际是`破坏佛法`的，没有道理你破坏佛法还保佑你，保护你不就是保护你破坏佛法吗？），凡是跟他有善缘的主命鬼王就说，他会先命令在这个地方的鬼神，每个地方都有鬼神负责管理这些小鬼的，负责管理这个地方的，不是说大城市没有，哪儿都有的，只是我们看不到的，命令这些土地神，宅神来保护母子平安，使得生产能够安稳。
