# 地藏菩萨本愿经讲习 第一百四十四讲

> 校订: 李师兄，胡师兄，利行师兄 <br />
> 审核: 胡师兄，利行师兄，孙师姐，黄兰师姐 <br />

地藏菩萨本愿经讲习

感恩地藏菩萨加持！

> 何以故。是阎浮提人。初生之时。不问男女。或欲生时。但作善事。增益舍宅。自令土地。无量欢喜。拥护子母。得大安乐。利益眷属。

前面主命鬼王说，众生**不会我意**，所以导致众生生时，死时都不安宁，补充一点就是说，众生为什么不会主命鬼王的意？也就是说为什么不听从主命鬼王的安排？其实主要还是因为`无明`，对于因果报应的决定道理无知，`我见我执深重`的关系，你或者说，这个因果报应嘛，我怎么会不晓得呢？善有善报恶有恶报嘛，对的，但是关键在于什么是善？什么是恶？很多人说不清楚，说得清楚也未必办得到，因为无始以来的贪嗔痴太重了。乃至很多人明明在作恶，他会认为自己在行善，比如很多人杀牛宰羊祭拜所谓的鬼神，比如所谓的`宰牲节`，西方的`复活节`，中国的各种节日，他们认为不杀生的话讨不到好彩头，他们认为杀生祭拜就是所谓的`行善`，他们认为猪牛羊生下来就该给他们杀该给他们吃，甚至屠夫认为多杀几头猪，猪肉价格便宜点就是在行善。如`喇嘛教`等认为宣传`男女双修即身成佛`就是在行善，很多诽谤如来藏的也认为他们在行善，比如乐此不疲的`ID南伽他`这个人？乃至`普德海幢`，`琅琊阁等诸人`破斥正觉也认为自己`善莫大焉`（正觉管理即便有问题，但`正觉的法义绝对没有问题`的，我对正觉的一些安排也有意见，但我从来不说法义的问题，因为佛法是这样的）。所以说你晓得因果报应不是问题的关键，问题的关键在于无明所以导致什么是善什么是恶搞不清楚！往往会善恶颠倒，这样就是不会我意的根源所在，而这个根源之所以产生就是因为`无明`，也就是`愚痴无智`。

然后主命鬼王接着说，是先说正确的做法，为什么要先说正确的做法呢？这样体现了他的`大慈大悲`和`高超的智慧`，`先建立正确的`，正确的应该怎么样做，然后`再叙述众生错误的做法`，`以及这些错误的做法会导致的严重后果`。`最后再结成正确的做法之必要性`，这样就能够给读者以强烈的印象，你如果先说错误的做法，最后再结成正确的做法，相对来说提醒正确的做法之必要性就要降低很多的。所以我写了个专栏命名`佛教文学`，佛教经典文学性是非常强的。

首先主命鬼王这位示现为鬼王的大菩萨就说，我为什么说阎浮提的众生生时死时俱不得安？这里先讲生的时候应该做的正确的做法，本来应该说在孩子出生的时候，不论是他的父母还是爷爷奶奶等，或者说准备要生小孩的时候，或者还在预产期的时候，孩子还没有出生的时候，就应该为这个即将要出生的孩子做种种善事，这种种善事一般人看到这里会想，比如世间法上的善事应该也是善事吧？也是，但这里的善事首先是`佛教经典`说的`善事`。

`佛教`的`善事回向`的`功用才大`，比如在本群有一些善友对于这一点还是不明白，说一个不相信三宝的人给他回向他也能够得益，实话说得不了多少的。以前还有人认为，一位不信三宝不信净土的，回向的话也能够超度到`西方极乐世界`，这是不可能的！因为极乐世界不是强拉强拽谁去，还要他`本人有这个心愿`才行，如果他都不信有极乐世界不愿意去，你再怎么样回向都是没用的，俗话叫做`不上道`的。如前面说做功德，**生者得六分，亡者得一分**，为什么要大声的在将要临终的面前念出来？也是要他自愿的，不说皈依三宝，至少在临终前对于三宝生起信仰，不然的话所谓的回向给他功德等，他得到的实际利益`微乎其微`的。所以在生的时候，如果你跟他有深缘，就要使得他最起码`对于三宝，对于佛教有决定信`，这一步很关键的。

所以这里的善事重点是说的佛教的善事，包括印刷善知识的著作，`平实导师`，`太虚大师`的著作努力的传播，这也是`法布施`啊，乃至为佛菩萨塑像，一个人没有多少钱和大家凑钱为佛菩萨塑像，乃至说修建寺庙出一份钱，诵读经典念佛等等，这样对于孩子已经出生还是即将要出生的都是有很大的功德的。

其次是说世间上的一些善事，比如`照顾鳏寡孤独`等，`修桥补路`等，如世间照顾鳏寡孤独等，这些人未必信佛但也是善事，世间人对于善恶的界定固然是模糊的，但也有正确的一些（到底什么是善什么是恶是根据结果来说的，如果现在得善报我们说当初造善因，现在得恶报我们说当初造作为恶因）。学佛人更要做这些事情，你有正见`在正见的基础上`去做这些事情，不单单说有福德还有功德，所以也应该提倡的。

这里主命鬼王说**增益舍宅**，所谓增益舍宅有法师说的舍宅都有所谓的宅神，增加宅神的力量这样讲也对，因为鬼神道跟我们的世界是重合的嘛（四天王天和四小天下是重合的），只是所见未必一样的。这里说增益舍宅更重要的说，对于这个家庭每一个成员来讲，每一个成员不管是七八十岁的老父母还是夫妻两口子，还是小孩都有益处。所谓的舍宅是说的家庭，家庭以人为本，每一个人的福德增强了增加了，这个家庭更加的幸福，每一个人更加的幸福当然是增益舍宅，过去所谓那些`五世同堂`的大家族，往上数好几代，大多都干过修桥铺路，为佛菩萨塑像，乃至灾荒的时候建粥棚免费布施饭食的事情。一个家庭善行越多，每一个人的福德越多，这个家庭才能够家大业大，要是不这样，一天到晚这个出问题，那个出问题，之所以家庭成员这个健康不好，那个工作事业不顺，说白了不就是没有`福德`嘛！这个家庭人人没有福德过不了多久就分崩离析了，所以这里的增益舍宅是增益每一个家庭成员福德的意思，孩子要出生或者已经出生，爸妈或者爷爷奶奶为他做功德，同时也增益了家庭每个人的福德，故名`增益舍宅`！

这里说**自令土地无量欢喜**，我们讲人类做善事，为啥`土地神`或者他所在的区域的鬼神欢喜呢？人类做善事，他不说出来其他的人大概是不晓得的，也看不出来他有福报还是没福报的，福报两个字又没有写在脸上对不对？而鬼神是完全看得到的，有福报的和没有福报的人所发出的光都不一样的，管理某个区域的鬼神和其他的鬼神（有些鬼神是属于某个地方常住的），他们之间不是按照钱财的多少来衡量高低的，因为他们早已经脱离了钱财的低级趣味了（钱财对他们没有任何的意义），他们是依据管理区域内的人也好乃至他管理的小鬼神也罢，`福德多少来衡量地位的`。比如`非洲冈比亚`的总统就是没有`欧洲比利时`的总理有地位，虽然两个国家面积差不多，但是冈比亚人人都穷，比利时人人有钱，鬼神道也是如此的，他管理的区域内人人有福德，或者很多人有福德，他在鬼神道的地位也不同，所以这里讲，**自令土地无量欢喜**，这跟他是息息相关的。

你想他的区域内，某一家人因为要生孩子做种种功德，好比世间某个地方，某个县某个市，某人在那个县长市长管理的县市开了家大公司，他作为地方领导，他能不欢喜吗？`鬼神道也是这样的`，你因为要生孩子的关系做了`弘扬正法`乃至`布施三宝`的大福德，在他的区域还有这样的善人他很开心啊，也意味着他的地位的提高，所以他肯定要来拥护子母确保平安啊（就好像某市招商引资成功，大企业入驻市长也要亲自接待安排优惠条件，最起码促进就业增加税收），那么就因为这样的道理，所以家庭的福德得到了提高。并且子母还能够平安并且家庭家大业大，这样是不是得大安乐呢？肯定是得大安乐嘛，总而言之将来这个孩子是你的眷属，你去做这些功德，不单单利益自己，还利益这个孩子，以及你老公老婆或者父母等所有的家庭成员。
