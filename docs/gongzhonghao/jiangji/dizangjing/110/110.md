# 地藏菩萨本愿经讲习 第一百讲

地藏菩萨本愿经讲习

感恩地藏菩萨加被

> 普广，汝以神力，遣是眷属，令对诸佛菩萨像前，志心自读此经，或请人读，其数三遍或七遍。如是恶道眷属，经声毕是遍数，当得解脱，乃至梦寐之中，永不复见。』

出现这样的梦境，绝大多数人是没有什么想法的，主要是因为纵使学佛了...不学佛的`断灭论`，`信基督`的认为上帝创造一切的这些我们且不说，单单就信佛的人来说也未必就会认为梦里面出现某人愁悲哭啼等是自己今生逝去的亲友乃至过去世的眷属，是希望他想办法救拔苦难。所以佛世尊怜悯一切众生，就对普广菩萨说，以你的神力让做这种梦的人，使得他们意识到这是过去世跟自己有深缘的眷属在请求帮助。

首先应该看得出来，这`普广菩萨`绝对是`八地以上`的`大菩萨`了，大菩萨可以和有缘的一切众生的`如来藏`直接感应的，所以他晓得你做了什么样的梦的，晓得你是什么样子的处境，并不是我们这样需要打个电话，要不发个微信，喂喂，你昨晚做了个什么梦？他直接可以和你的如来藏感应，能够感应到你的`如来藏中种子的流注`，那个梦不也是种子流注产生的嘛。

所以这里世尊要对普广菩萨摩诃萨说，你以你的神力，令这些做这样的梦的人，怎么样怎么样，很多人可能也不信，很多事情你没有经验过，你肯定不信。就说我这次讲地藏经，上次我讲到`地狱名号品第五`的时候，我讲`普贤菩萨摩诃萨`的名号之含义，我晚上就做了个梦。

就梦到一位十六七岁的`美少年`（因为菩萨是现天人身，天人一生数万年看起来都像十六七岁的少年或者少女的，他们没有老病可说），其实我今年（`2020` 年）因为我个人的感情生活很惆怅的，跟我生活六年多的人离婚了（她是我在KTV认识的，我当初在台湾人的公司卖食品添加剂全公司就我业绩最好，台湾老板喜欢唱歌也请顾客去KTV，看我业绩做得挺好经常叫我一起作陪，我也是拒绝的，但拒绝的次数多了也不好，于是在一家KTV遇到我前任，她被吸毒的所谓男朋友控制在夜场赚钱，赚不够钱经常挨打，有几次我都看她手臂清淤，后来就带她跟我跑了（当初社会治安也不好），结果不到七年出轨两三次。

2020年这次出轨是真的跟人跑了（后来又想回来，我拒绝了），没有扯离婚证之前还各种方式骗我钱，说自己生病又是爹妈生病，病好了就回来，不到半年我起码打款接近二十万，还骗我今天江苏，明天四川的去接她，路上我还遇到车祸了，甚至和他男的准备埋伏我，再敲诈我一笔）。这真是让我很狼狈和愁苦，我讲地藏经第五品就前段时间就梦到这位`美少年`他就说"`你妹妹不是也交了三个男朋友才结婚的吗`"（这我绝对不知道的，因为她大学毕业就去新疆支教了，我晓得加她老公就两个）？我一时语塞，他看看我就笑笑说"`没事的，没事的`"！我觉得他很可亲，很善良豁达的，就问了您是谁？他说，你认识的！然后他又说，你右边脸上有一道痕迹不好，来！我给你抹去！就用手指头给我抹掉，抹去之后我感觉脸上很烫，后来我就跟他在一个美丽的公园转转，我感觉他挺好的，我就要求加他微信，他非常爽快的答应了，再后来我就醒来，感觉右边脸还有明显的烫烫的感觉，第二天早上我问我妈，我妹是不是交三个男朋友才和妹夫结婚的？我妈说，是的啊！新疆支教还有一个的。我才想起来，我昨晚上在群里讲地藏经的时候讲到`普贤菩萨`，来到我梦中的应该就是`普贤菩萨摩诃萨`了，不然怎么会这么巧啊？并且我也因为这个梦不久就放下了这段孽缘，彻底的解脱放下了这段孽缘（按照命理分析由于先世业报由此导致残疾等灾祸也化为无形了）。

我也就是说，大菩萨的第八识可以和一切众生的第八识感应的，这样就可以知道一切众生干过什么事情，甚至是你做过什么梦都知道的。`佛陀`大慈大悲所以要`普广菩萨`让产生这样的梦境的人去作这样的功德，怎么样的功德呢？就是说`与地藏菩萨摩诃萨的悲心愿心相应`，最起码要对地藏菩萨的功德大愿有`了解赞叹`，对于地藏菩萨有母子之间的`恋慕`，乃至`发普度众生的大愿`（当然最好是有`正见`，没有正见度什么众生呢？）这样叫做志心，志心的在诸佛菩萨像前，自读这本地藏菩萨本愿经，要是自己不识字，过去也有很多人不识字的就请人来读。或者自己读三遍或者七遍，那么自己堕落到恶道中的眷属，不管是地狱道还是饿鬼道还是畜生道，这样读三遍或者七遍之后，就能解脱恶道往生善道，也就是往生人道或者天道，以后在梦中就不会再出现这样的梦境了。

那么这里有个问题，既然佛陀大慈大悲，佛陀当然也晓得这些事情，为何佛陀不直接救拔这些苦难的在三恶道的众生？而要他的眷属也就是你去干这个事情？因为你跟目前在三恶道的那个众生`有深缘`，虽然说无始以来众生都在三界中流转，可是世界无数，国土无量，众生是无量无数，其中从`缘分`来讲`有亲有疏`甚至有`无缘`，所以要你去作这样的功德救拔他，他未必和佛陀直接有缘当然就没有办法，只能通过你的缘而使得他和佛菩萨结缘。

佛世的时候有个`东城老母`，就在舍卫国东边住。舍卫国基本都信佛的，这个东城老母却不信，凡是比丘走那里过就嫌弃得很，后来比丘们就讨论这个事情，就说佛陀这么大的智慧，这么大的神通肯定能够度化她的，佛陀说不行，她跟我没有缘，大家都不信。佛陀就说，那么我去吧。傍晚的时候，佛陀就专程去给她讲法，去之后就先给这个东城老母讲佛理，东城老母就把耳朵塞住`不听`，这讲佛法看来是不行的。然后就示现神通，佛陀升到几十丈高，放出金色光芒，照得就好像白天一样的，东城老母就用手把眼睛遮住。佛陀运用神力，使得她的十个手指头都出现佛像，她就`把手背过去`，`把眼睛闭起来`...佛陀说，不行，我晓得有个人行，`阿难`！你过来，你来跟她讲法，说来奇怪，阿难尊者一来，没讲几句，这个东城老母就信佛了，愿意供养阿难。

所以为什么要让普广菩萨以神力遣是眷属，令对诸佛菩萨像前自读或者请人读这本地藏菩萨本愿经呢？佛陀是有深意的，是有道理的，就是这样的，所以经典你看起来好像没有什么，其实里面都包含了深意以及佛菩萨的大慈大悲的。

> 复次普广：『若未来世，有诸下贱等人，或奴或婢，乃至诸不自由之人，觉知宿业，要忏悔者。志心瞻礼地藏菩萨形像，乃至一七日中，念菩萨名，可满万遍。如是等人，尽此报后，千万生中，常生尊贵，更不经三恶道苦。』

然后世尊接着对普广菩萨摩诃萨说，如果在未来世，有这样的情况有`下贱`的人，这里的下贱不是说思想上行为上的下贱人，是说的工作生活上的，比如`卖苦力`的，比如说一天又累又苦又没有地位的，给人家当佣人的当仆人的，搬砖的送快递的种地的农民工的等等。乃至说干一些难以启齿的工作的，乃至说`不自由`的人，自由有财富上的自由，一个月累死累活也就千把块钱，肯定不自由，乃至说行为上的不自由，比如生病的人有重病的，怎么自由呢？对不对？想去哪儿都去不了，肯定不自由的，不自由有很多种的，当下这个阶级唯物论占主流的时代，网络上一大伙人喜欢说，你为什么不当马云？你为什么不当李嘉诚？你偏偏要去工地上搬砖！？这些人真是愚痴得一塌糊涂，古话说得好，**赚大钱看命，挣小钱靠勤**。没那个命能怎么办呢？这辈子是上辈子造业的一个`异熟果`，上辈子种白菜，这辈子收获白菜，你偏偏要他收获腰果，他也想啊，三十多块钱一斤，谁不想啊？问题是`上一季没种`啊！

很多人看到佛教讲命讲先世业报就认为`佛教是宿命论`，其实`不是`的！佛教认为`今生果是过去世的因`，但`未来果是现在创造的`，而宿命论是说过去现在未来都是决定的，也就是说佛教不认为未来是决定的，未来是你现在创造的！而且现在也不是决定的，虽然你我生来就是这么个大环境就是这么个爹妈，我们当然不能把老爹变成李嘉诚，我们就是这样的农民老爹，这是我们的命！但同样的机遇不同的人格，比如穷是业报导致的，也有贫贱不能移的也有因为穷而作奸犯科的，人人都有如来藏，`如来藏生前面七识`，`如来藏中种子无量无数`，就说人人都有`主观能动性`的，心理上有无数的可能，未来也有无数的可能的。

所以这一段经文说，**觉知宿业**，就是说相信因果报应。看到这一颗白菜，知道自己过去种的肯定是白菜，这样叫做觉知宿业，并不是说一定要有宿命通。你看到这颗`白菜`，你不知道三五月前肯定种植的白菜种子？不能像网上那些愚痴人一样的，你为什么不收获腰果？你再怎么样努力也收获不了腰果啊，没有种啊，种的是白菜啊！否定因果轮回那些愚痴人是不会觉知宿业的，如果没有特殊情况，将来在三恶道中要流转很久很久的。我们不是否定人不应该勤奋不应该努力，我们是说在勤的努力的基础上你能够怎么样，实际上是在命这个基础上的（所谓家庭环境，社会环境等等都属于`如来藏共变自变`是`性境`的，`性境是不随心的`！我们的努力在这个基础上的，譬如鱼儿可以游得快多吃虫，可你怎么游也是在这条`河`里的）。比如你种白菜，你多勤奋一点，你可以收获很多白菜，但是白菜永远都不是腰果，也许你可以收获个几百公斤等于人家的三斤腰果，我们不反对人勤奋，我们是说这确实有个命的前提。

那么对于自己目前的情况有个清醒认识，并且深信因果轮回因果报应的人，他这辈子虽然不怎么样乐观，但是他有这个正确的知见就会`皈依三宝`，皈依三宝之后就`至诚的忏悔`，我这辈子这么苦肯定是因为上辈子的某种恶业造成的，或者说没有造善业造成的。那么就要忏悔，忏悔要在地藏菩萨摩诃萨的像前忏悔，在一天到七天之中，念地藏菩萨摩诃萨的名号满万遍，其实这样就是`拜地藏忏`，拜地藏忏在一天或者七天之中吃素，`日中一食`，就好像出家一样（或者本身在寺院进行也可以），`中夜休息四个小时`，其他时间都在精进的`念地藏菩萨名号`，深心忏悔满万遍圣号，这样满万遍之后，这辈子这个异熟报结束了，千万生中命运就比较好，就会生来尊贵，就算有其他业的影响生来不尊贵，后面也会尊贵的（比如`朱元璋`那样生来穷困潦倒，后来大富大贵），常常生在尊贵的家庭中，千万生中都不会下堕三恶道。

末法时代很多人不信因果报应，这就糟糕了！他们认为我今天给菩萨像供了几根香蕉，我明天买彩票就应该中五百万，明天不行我后天也可以。或者说`学喇嘛教密宗`的那些人，我念了这个`伪咒`，或者这个咒语就应该怎么样怎么样，实际上这辈子是上辈子所种的一个`异熟果`（我翻来覆去的讲这一点，因为不清楚这一点决定要退失学佛，迟早的事儿）。这个异熟果出现的时候，就体现了上辈子的业种在实现，是很难改变的，不可能把白菜变成腰果，诸佛菩萨也只能够说让你的白菜多卖两块钱，或者你本有善因，要晚点出现，让这个善因早点出现，又或者让小善因得善报增长广大。你再怎么样，也不可能把自己的老爸改成王健林，你又不姓王，对不对？真正的经典都是建立在`如来藏受熏持种因果报应`的基础上的，所以那些人是愚痴人，你供养几根香蕉，你没有中五百万，就要诽谤佛菩萨？你真正应该找原因，找自己的原因。上辈子所种是你自己种的啊，又不是佛菩萨帮你种的，对不对？

谁种植谁收获嘛，都是在自己心地如来藏种的，不就自己收嘛，这能怪谁呢？所以网络上那些诽谤佛法的愚痴人，真是让人哑口无言，正确的做法不应该像他们这样谩骂，我好心好意告诉你，你自己种植的是白菜（上一世没有造善业或者上一世有恶业这辈子实现了），你还骂我？我让你现在吃白菜糊口的时候，改种腰果（现在赶紧培植福德），你还觉得是我种植了白菜让你只有白菜可吃的？正确的做法应该是`好好的忏悔`，人都有无穷的未来世的，`不要怨天尤人`，更`不诽谤因果轮回诽谤三宝`，而`应该忏悔改过`，现在`善言善行`，都有常住真心如来藏的，你真能这样作，未来不是也可以**他日我若为青帝，报与桃花一处开**吗？

> 复次普广：『若未来世中，阎浮提内，刹利、婆罗门、长者、居士、一切人等，及异姓种族，有新产者，或男或女，七日之中，早与读诵此不可思议经典，更为念菩萨名，可满万遍。是新生子，或男或女，宿有殃报，便得解脱，安乐易养，寿命增长。若是承福生者，转增安乐，及与寿命。』

世尊接着对普广菩萨说，如果在未来世。`阎浮提`是什么意思？我前面讲过的，还记得吗？阎浮提的正确翻译是说的`胜金`，就是说这个世界过去现在未来，都以`黄金`为`硬通货`，比如美元就是跟黄金挂钩的，这是什么世界？这就是地球，在这个苏弥卢系太阳系中，四小世界有人类的世界，只有地球是这样的以黄金为本位，郁单越根本没有交易这回事情，其他也不是把黄金当硬通货的。就是说我们这个世界的人类。刹利就是刹帝利，在印度属于政府阶层。婆罗门是祭祀阶层，这两者都有尊贵的社会地位。在我国来讲也可以说有社会地位的比如公务员或者学者，乃至说社会地位比较高，经济地位比较高的为长者。长者受人敬仰的，居士在家学佛的。乃至一切人等等，一切人等就包括了异姓种族，古印度是把人的阶层分为四等的，`婆罗门`为第一，`刹帝利`为第二，然后是`吠舍`和`首陀罗`，吠舍是手工艺人，首陀罗是农民，还有大量的贱民达利特从事低贱的活儿，这里异姓种族就包括了一切人，在我们国家也可以说三教九流的人都包括进来。
