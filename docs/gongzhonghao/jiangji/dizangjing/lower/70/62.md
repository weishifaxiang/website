# 地藏菩萨本愿经讲习 下 第六十二讲

> 校订: 李师兄，胡师兄，利行师兄 <br />
> 审核: 胡师兄，利行师兄，孙师姐，黄兰师姐 <br />

地藏菩萨本愿经讲习（嘱累人天品一2021.7.10-2021.7.16）

我们现在正式进入最后一品，就是“嘱累人天品第十三”

这里说嘱累，其实这是两个词组合而成的，所谓“嘱”佛说的称为付嘱，佛付嘱我们学习大乘的菩萨们。就是佛金口宣说，对我们有付嘱，我们应该怎么样。“累”是说的付嘱的内容是叫我们要宣传大乘经典，宣传地藏菩萨本愿经。这是一层意思。这是从佛旨来说。

第二“嘱”是说的顶受的意思，“累”是说的心甘情愿的愿意这样去做，这是从菩萨的角度来讲，我们这些小菩萨倘若不这样去做就没有办法积累广大的福德和智慧，所以我们应该顶受，要宣说地藏菩萨本愿经，我们是完全至诚，是完全心甘情愿的这样去做的，原因其实就是自利利他。从嘱来说是自利，我们宣说宣传地藏菩萨本愿经增进我们的福德智慧，所谓善有善报颠扑不破，从累来说是从利他来讲，我们宣说地藏菩萨本愿经能够使得更多的苦难众生迈向三乘菩提，究竟安乐，这是从利他来说，自利利他是从菩萨角度来讲。

第三，“嘱”是佛陀金口宣说，“累”是菩萨一片丹心顶戴受持，因佛而有佛所讲法，因法而有菩萨僧，嘱累就是三宝。这是把前面两者合起来说。

接着说到嘱累人天品，先说到“人”，六道众生，为什么只说人天，不说阿修罗？不说畜生道的？不说地狱道的？不说饿鬼道的？而只说人天？因为人，两笔为人，立于天地之间，没有老虎的力气没有豹的速度，人之所以能够立于天地之间，是因为人相对于其他道的如前面广说，有更强的造作性，有思想，主观能动性，能够善思惟、善筹量、善忆念，正因为人能够思想所以可以筹量可以计算可以产生语言，可以产生丰富的语言所以可以忆念，在这个基础上我们可以相互交流，可以创新发明，可以把思想通过纸张，通过芯片保留下来，超越时间空间的局限性，不会随着我们的逝去而逝去，更不会随着我们在某地而不能够传播到其他地方其他区域。这就是因为我们有如来藏，有如来藏根据人这个业报导致我们的前面六识具备这样的功能性，创造性，而畜生道，地狱道，饿鬼道业报的关系显然就没有办法达到这一点，所以首先要说人，嘱累人。

另外地狱道太苦，太苦的缘故难发道心，不间断的受苦如何学佛？饿鬼道饥虚，大多数的饿鬼都没有吃的，也学不进去的，畜生道太愚痴，也就很难闻思修了，人道如前面说具备闻思修的条件，再者人道苦乐参半，即便是乐受也有行苦，但对比三恶道是苦乐参半，可以知苦断集，知苦是学佛的首要条件，又能闻思修，当然人道为道器。

接着为什么要说“天人”呢，对比人类来说天人是没有人类勇猛精进的，他们的日子过得很好很愉悦，根本就没有什么苦可言，虽然说有五衰现前，总而言之这不过就是死苦而已，在他们漫长的寿命中不会有我们的这样的病苦，老苦，求不得苦等等，所以他们修行佛法是不会有人类这样精进的，不苦谁求解脱？自己不苦大多也不会在意其他人苦不苦！所以有些地方说天人难发道心，主要是从天人境界是胜妙来讲。

其次天人对比我们结使就更轻微，五钝使五利使就更轻微，人之所以贪，除了本身有贪的种子，更重要的是缘也就是条件会让烦恼种子现行，比如你每个月还房贷五六千，你一个月才七八千，你能不贪吗？而天人不存在这个问题，自然产生种种房舍，种种饮食，所以没有必要去贪。更何况他们本身就是修十善业道才能够往生到那里去的。

正因为天人结使薄，贪嗔痴薄，寿命又长，所以他们如果真正学佛，从这个角度来讲是很容易证果的，我们这个人类世界很多人可悲就在于等到某天想学佛了，无常大鬼也就来了。所以因为以上的原因，我们的佛陀嘱累人道的，天道的要广宣地藏法门，并且嘱咐地藏菩萨摩诃萨加持，普度人道和天界的众生。

那么你会说，不是还有阿修罗道吗？阿修罗道是在其他的五道中的，他们没有单独成为一道，天界的阿修罗其实就是一些天人，只不过因为嗔心重长相丑陋，受用对比其他天人有差距，因为嫉妒乃至嗔恨重所以对其他天人产生极大仇恨。人道的阿修罗就是如希特勒，斯达林，某某某，波尔布特那样的，反正就是天斗地斗的，人类刀兵劫来了，他们就会出现，然后就是尸山血海。鬼道的阿修罗是一些大力鬼王之类的。畜生道的阿修罗有三种，一种是陆行的，一种是在水中的，一种飞行的，飞行的相貌丑陋长着黑色的翅膀很庞大，假如你看见了就把他当怪物。陆行的是在荒凉的深山老林里面的，水中的白天在大海的低云层中休息，晚上才在海中，说海中的能够变成大海怪掀翻船只，一般人看不到（畜生道的阿修罗大多也能变化，他们的福德比人类更比其他畜生道的大，生而有报通），看到也都吓得半死……至于地狱道有没有？有些说法是有的，不过无间地狱等没有，小地狱有是专门折磨罪人的。所以阿修罗道不单独列出来。

经典中有人曾经问过佛陀，可以不可以发愿，令五趣众生都可以成佛（阿修罗包括在其他五趣中）？也就是说可以不可以以鬼道成佛？以畜生道成佛？以地狱道苦生的形态成佛？佛陀说，不可以的，因为除开人道和天道，其他都属于非器之身，如果要让他们成佛，首先应该把他们化度成为人天的形态。然后才能够使得他们闻思修，才有成佛的可能。所以如地藏菩萨摩诃萨等在三恶道示现，也是使得他们先离开三恶道，到人道或者天道中来，再使得他们闻思修，戒定慧而学佛。之所以三恶道不行，是因为业报的关系，三界之中没有任何一个众生他不是如来藏持种，种子生现行而有的业报身（修禅定业往生色界无色界天也是业报身），三恶道一样是业报身，它是那个业报就没有丰富的表义名言，就结使很重，环境很差，没有办法学佛，只能够使得它们先离开那里，往生人天，然后才能勤求佛道，所以这里说嘱累人天品，没有说嘱累五趣品（比如一头猪还不是饿鬼道的没有吃的，猪什么都吃的，更不是地狱道那种时时受苦，请问即便你不顾爸妈的谩骂，不顾邻居的非议，你把它弄到家里来养着，你怎么让它学佛？让它念佛乃至读诵经典？）。

裴休丞相（裴休你没有听说过，白蛇传的法海法师你总是听说过吧？他就是法海的父亲！只不过白蛇传是瞎扯，他根本没有拆散许仙和白娘子。法海实际是一位高僧，是替代皇子出家的，他的确是有禅定神通，说以前在寺庙那座山腰有个洞，洞里有一条大蛇，来拜山的多被大蛇袭击，他驱逐了那条蛇，那条蛇是真的白色的大蟒蛇）云：“呜呼！生灵之所以往来者，六道也。鬼神沈幽愁之苦，鸟兽怀獐狨之悲，修罗方嗔，诸天正乐，可整心虑，趣菩提唯人道为能耳。”

裴休在唐代是做宰相的，他就是说，所谓生灵各种形式不外乎就是六道，六道来讲鬼道等沈幽愁，所谓幽愁主要是清虚，大多数鬼道的都没得吃的饥虚得很（这个其实不因为你施食而有根本的改变，好比人间的苦就是求不得苦，假如有天人愿意帮助你，你我也很难做到事事如意的，因为我们是人，从投胎成为人的那天开始，哪怕是皇帝都有种种求不得苦！鬼道也是一样，很难改变他们时时饥饿的状态，除非他们死掉了投胎到其他道中）。鸟兽怀獐狨之悲，獐是说的小动物，狨是说的猴子，这些动物都太蒙昧了，没有办法教导它们懂得诸经大义。阿修罗嗔恨很重，比人类嗔恨心重很多倍，要他们放下嗔恨来学佛也很难（嗔恨重其实也就贪爱重，嗔恨和贪爱重也就意味着愚痴重，不是说人间修罗不会打仗不会发明不会算计不会整人害人，这些他们精通得很，但是出世间的很难契合，而且绝大多数的阿修罗下一世是决定下地狱的）。天界又太舒适太愉悦了（就是我前面说的天界难知苦，他几乎没有苦，所以难发道心），要像人类学佛这样精进难以做到，所以可以“可整心虑，趣菩提唯人道为能耳”所以综合来说，还是人类堪为道器！我们是人千万别辜负了这个人身。
